Taskly - Modern Todo List Web App

A modern and responsive Todo List web application built using HTML, CSS, and JavaScript.  
Manage daily tasks with priorities, due dates, filters, statistics dashboard, and a clean user interface.

✨ Features

- ✅ Add new tasks
- ✅ Edit existing tasks
- ✅ Delete tasks
- ✅ Mark tasks as completed
- ✅ Priority system (Low / Medium / High)
- ✅ Due date support
- ✅ Notes section for tasks
- ✅ Task statistics dashboard
- ✅ Filter tasks by status
- ✅ Responsive design
- ✅ Smooth UI animations
- ✅ Modern clean interface
- ✅ Local storage support
